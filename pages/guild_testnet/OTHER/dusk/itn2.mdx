## **Open Ports**

```bash
sudo ufw allow 22
sudo ufw allow 8080/tcp
sudo ufw allow 9000:9005/udp
sudo ufw enable
```

## **Install Github**

```bash
curl --proto '=https' --tlsv1.2 -sSfL https://github.com/dusk-network/itn-installer/releases/download/v0.1.0/itn-installer.sh | sudo sh
```

## **Import Wallet Phrase**

```bash
rusk-wallet restore
```

Paste Your Phrase (Make Sure All Lowercase)

Enter Password Twice

## **Run the Code Below**

```bash
rusk-wallet export -d /opt/dusk/conf -n consensus.keys
```

Enter Password Twice Again

```bash
sh /opt/dusk/bin/setup_consensus_pwd.sh
```

Enter the Same Password Again

## **Start Node**

```bash
service rusk start
```

## **Check Node Log**

```bash
grep "block accepted" /var/log/rusk.log
```

or

```bash
tail -f /var/log/rusk.log
```

## **Check Wallet**

```bash
rusk-wallet
```

- Choose Access Your Wallet and Enter
- Enter Password (Ensure Your Address Matches the One on the Website)
- Use Up/Down Arrows to Navigate Menu and Enter to Execute
